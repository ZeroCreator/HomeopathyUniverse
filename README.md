# Ковчег гомеопатии

Интерактивная периодическая таблица элементов с соответствием растений (по Яну Схолтену) и животных.

## Возможности

- **Периодическая таблица** на CSS Grid с 18 колонками и 7 + 2 рядами.
- **Фильтрация отображения**: химические элементы, растения, животные или всё вместе.
- **Клик по ячейке** открывает модальное окно с полным описанием.
- **Отдельная страница** для каждой ячейки (`/cell/<period>/<column>`) — можно поделиться ссылкой.
- **Раскрывающиеся описания** колонок и рядов.
- **Поиск** по элементам и свойствам.

## Технологии

- React 19 + TypeScript
- Vite
- Tailwind CSS
- React Router DOM

## Установка и запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Структура данных

Данные таблицы хранятся в `src/data/tableData.json` и содержат:

- `columns` — 18 колонок с названиями и описаниями.
- `rows` — 9 рядов (7 периодов + лантаноиды + актиноиды).
- `cells` — массив ячеек с символом элемента, названием, свойствами, растениями и животными.

Данные извлечены из PDF `Таблица цветная.pdf` с помощью скрипта `extract_all.py`.

## Деплой

Рекомендуется развёртывать на Vercel, Netlify или GitHub Pages.

```bash
npm run build
```

После сборки папка `dist/` готова к публикации.
